# Healthcare Bot using Dialogflow Essentials

## Description
This project is a Healthcare Chatbot developed using Dialogflow Essentials.
It helps users with basic healthcare-related queries such as symptoms,
precautions, and general health advice.

## Technologies Used
- Dialogflow Essentials
- Google Cloud Platform
- Node.js (for webhook fulfillment)

## Features
- Answers healthcare FAQs
- Symptom-based responses
- User-friendly conversational flow
- Webhook integration (optional)

## How to Use
1. Import the Dialogflow agent ZIP file into Dialogflow console
2. Enable fulfillment (if required)
3. Deploy webhook using Node.js
4. Test the bot using Dialogflow simulator

## Project Purpose
This project is created for academic/learning purposes.
